## How to use

### Browser

```js
<script src="lib/WebModule.JSZip.js">
<script>
console.log( WebModule.JSZip );
</script>
```

### WebWorkers

```js
importScripts("lib/WebModule.JSZip.js");

console.log( WebModule.JSZip );
```

### Node.js

```js
var WebModule.JSZip = require("lib/WebModule.JSZip.js");

console.log( WebModule.JSZip );
```
